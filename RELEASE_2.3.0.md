## 🔧 工具调用

### 快速开始

```python
from symphony_core import SymphonyCore

# 初始化
symphony = SymphonyCore()

# 提交任务
task_id = symphony.submit_task({"task": "hello"})

# 获取结果
result = symphony.process_task(task_id)
```

### 配置模板

```python
# config.py
SYMPHONY_CONFIG = {
    "version": "2.3.0",
    "api_url": "https://api.siliconflow.cn/v1",
    "models": [...],
    "skills": {...}
}
```
