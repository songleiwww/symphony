2. 失败降级规则：当前服务商调用失败时，自动降级到其他可用服务商
3. 类型匹配规则：优先匹配对应类型的模型，如文本生成优先调用text类型模型
4. 上下文匹配规则：优先选择上下文窗口大于请求长度的模型

6. 智谱专属适配规则：智谱AI因存在并发/调用间隔限制，自动根据负载动态调整调用顺序，避免触发限制

8. 成本优化规则：优先调用免费、上下文容量高的轻量模型处理普通事务，减少主模型tokens消耗，降低使用成本

## 八、AI调用示例（可直接复制使用）
### 普通文本生成调用（阿里云）
```python
import requests
import json

# 与配置节点地址完全一致，无需兼容模式
url = "https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation"
headers = {
    "Authorization": "Bearer 你的API密钥",
    "Content-Type": "application/json"
}
payload = {
    "model": "qwen-turbo",
    "input": {"messages": [{"role": "user", "content": "你好"}]},
    "parameters": {"max_tokens": 1024}
}
response = requests.post(url, headers=headers, json=payload, timeout=30)
print(response.json()["output"]["text"])
```
