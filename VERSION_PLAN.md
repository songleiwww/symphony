# 序境系统 (Symphony) 版本规划

## 版本命名规则

采用语义化版本 (Semantic Versioning): `主版本.次版本.修订号`

- **主版本 (Major)**: 重大架构变更，不兼容的API变更
- **次版本 (Minor)**: 向后兼容的新功能
- **修订号 (Patch)**: 向后兼容的问题修复
