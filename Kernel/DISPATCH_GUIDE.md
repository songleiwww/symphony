## 调度模式

```
用户问题 → 任务分析 → 选择官署角色 → 每个角色使用自己绑定的模型 → 协作输出
```

## 示例

1. 用户问技术问题 → 调度"少府监"角色 → 使用Llama模型
2. 用户问视觉理解 → 调度"掌冶署"角色 → 使用Llama 3.2 Vision
3. 用户问中文对话 → 调度"中尚署"角色 → 使用GLM模型
