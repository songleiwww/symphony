# Kernel内核管理说明

## 内核目录结构
```
kernel/
├── sf_team_roster.json      # 少府监官属花名册（核心配置）
├── kernel_loader.py         # 内核加载器
├── config_manager.py        # 配置管理器
├── permission_manager.py    # 权限管理器
└── update_manager.py        # 版本更新管理器
```

## 内核核心文件说明

### 1. sf_team_roster.json - 少府监花名册
**核心配置文件，不可随意修改**
包含少府监51位官属的完整信息：
- 基础信息：ID、姓名、性别、官职、职务、描述
- 模型绑定：模型服务商、模型名称、API配置
- 权限配置：可调用工具、调度优先级、访问控制
- 专长领域：擅长的任务类型、知识范围

### 2. kernel_loader.py - 内核加载器
系统启动时自动加载内核配置：
- 加载花名册配置
- 验证API配置有效性
- 初始化权限矩阵
- 预加载常用模型配置

### 3. config_manager.py - 配置管理器
管理内核配置的增删改查：
- 模型配置的添加/删除/修改
- 角色权限配置更新
- 限流规则调整
- 配置变更日志记录

### 4. permission_manager.py - 权限管理器
细粒度权限控制：
- 角色权限验证
- API访问控制
- 敏感操作审核
- 操作日志记录

### 5. update_manager.py - 版本更新管理器
内核版本管理：
- 配置版本备份
- 增量更新支持
- 升级失败回滚
- 版本兼容性检查

## 内核管理规则
### ✅ 允许操作
- 新增/修改模型配置
- 调整角色权限和优先级
- 更新限流规则
- 新增官属角色

### ❌ 禁止操作
- 直接修改sf_team_roster.json原始文件
- 删除核心官属配置
- 未经审核修改API密钥
- 跨版本不兼容升级

## 内核版本控制
- 每次配置变更自动生成版本号
- 保留最近10个版本备份
- 支持一键回滚到任意历史版本
- 变更记录全程可追溯

## 内核安全
- 所有配置变更需要验证签名
- API密钥加密存储
- 敏感操作二次验证
- 访问日志永久留存
