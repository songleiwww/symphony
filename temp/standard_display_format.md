## 数据来源

| 列 | 来源表 | 字段 |
|----|--------|------|
| 角色 | 官署角色表 | 姓名 |
| 官职 | 官署角色表 | 官职 |
| 所属官署 | 官署表 | 名称 |
| 官署ID | 官署角色表 | 所属官署 |
| 模型 | 模型配置表 | 模型名称 |
| 引擎 | 模型配置表 | 服务商 |

---

## 正确SQL

```sql
SELECT 
    r.姓名 as 角色,
    r.官职 as 官职,
    o.名称 as 所属官署,
    r.所属官署 as 官署ID,
    m.模型名称 as 模型,
    m.服务商 as 引擎
FROM 官署角色表 r
JOIN 官署表 o ON r.所属官署 = o.id
JOIN 模型配置表 m ON r.模型配置表_ID = m.id
WHERE r.状态 = '正常'
ORDER BY r.id
```
