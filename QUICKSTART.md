# Symphony Quick Start - 交响快速入门

## 5分钟快速上手

### 1. 什么是交响（Symphony）？

交响是一个多模型协作系统，让多个AI模型一起工作，完成更复杂的任务。

---

### 2. 安装（30秒）

```bash
cd C:\Users\Administrator\.openclaw\workspace\multi_agent_demo
```

就这么简单！不需要额外安装。

---

### 3. 你的第一个交响（5分钟）

#### 运行天气播报站
```bash
python weather_final.py
```

#### 运行天机播报系统
```bash
python tianji_final.py
```

#### 运行模型自由讨论
```bash
python model_discussion_final.py
```

---

### 4. 核心概念

| 概念 | 说明 |
|------|------|
| **Model Chain（模型链）** | 17个模型按优先级排序 |
| **MCP Tools** | 工具调用框架 |
| **Memory System** | 记忆系统，记住上下文和偏好 |
| **Fault Tolerance** | 故障容错，模型失败自动切换 |
