# 序境 Symphony AI 内核

**序境** - 面向下一代人工智能协同操作系统的内核实现

## 架构设计

- **生产模式**: `18789` 端口 - 稳定运行环境
- **开发模式**: `18901` 端口 - 活跃开发调试环境
- **多模型调度**: 支持统一接口调度多种AI模型

## 项目结构

```
symphony/
├── src/
│   ├── core/           # 核心内核
│   ├── scheduler/      # 模型调度器
│   ├── gateway/        # API网关
│   ├── tools/          # 工具集
│   ├── memory/         # 记忆管理
│   └── plugins/        # 插件系统
├── config/             # 配置文件
├── scripts/            # 启动脚本
├── docs/               # 文档
└── README.md
```

## 启动方式

**开发模式** (端口 18901):
```bash
npm run dev
```

**生产模式** (端口 18789):
```bash
npm run start:prod
```

## 设计原则

- 留优去劣：持续重构，保持核心精简高效
- 功能性集中：相关功能聚合，模块化解耦
- 多模型调度：统一接口，灵活调度
- 开发/生产分离：稳定环境保障服务，开发环境快速迭代
